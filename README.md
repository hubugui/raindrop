raindrop
========

基于微信订阅号的网页链接分享服务

git和svn
========

初始化
--------
```python
* cd project home
* svn co https://your svn host/raindrop
* cd raindrop
* git init
* git remote add origin https://github.com/hubugui/raindrop
* git fetch origin
* rm -rf 1
* git merge remotes/origin/master
* svn commit -m 'merge git'
```
